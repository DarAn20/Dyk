# 🥋 Dang Yong Kwan — Sitio Web Oficial

Sitio web moderno y responsive para la academia de Taekwondo **Dang Yong Kwan Sang Sok Ja**, con sedes en Cartago y Atenas, Costa Rica.

## 🚀 Características

- Diseño moderno y profesional enfocado en conversión
- Integración directa con WhatsApp 📱
- Sección de sedes con mapas interactivos 🗺️
- Información de estilos:
  - Taekwondo ITF (Tradicional)
  - Taekwondo WTF (Olímpico)
- Galería de imágenes
- Botón flotante con selección de sede
- Totalmente responsive (adaptado a móvil)

---

## 🧱 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome (íconos)
- Google Fonts
